# vue-plugin-template

> Vue插件模板 |
> A template for generating vue plugin use Vue CLI 3.


## 使用 | Usage

```bash
vue init edwardorz/vue-plugin-template my-lovely-plugin

cd my-lovely-plugin

// dev
yarn serve

// build
yarn build

// build with report
yarn build:report

// generate docs
yarn docs
```


## 特性 | Features

+ 使用 `vue-cli-service build --target lib` 进行编译
+ 支持多种vue plugin形式: SFC/实例方法/全局属性、方法/directives/mixins
+ generate docs
+ less + pug
